快毕业了

发表于 2018-05-26 • 分类于 碎碎念

各自奔前程的身影匆匆渐行渐远

还有十多天就要答辩了，这样该是大学旅途的终点站了吧，突然想写篇长文记录下这个被我吐槽四年的母校渣校，这个超强辣鸡专业，但刚提笔就词穷了。我发现我就这个怪毛病，每想写点什么的时候却总是写不出来，若硬憋出一些话，读起来也是味同嚼蜡，就像你目前看到的这样，没错这段话也是我硬称的。。。

还有后续

这个作为新博客第一篇文章，本来是准备写毕业季的，但是提笔忘词，怎么写都不得劲，遂放弃。不过后面应该会写的，四年的生活，得容我慢慢回忆一下，好好写一下，然后发到朋友圈装个逼~ ヽ(✿ﾟ▽ﾟ)ノ

技能树

  • 目前主点 Java 后端开发，毕竟还得混口饭吃不是吗
  • 最近在研究 Chrome Extension，其实也不算研究啦，也就三三两两看了一点点，看完后希望自己能做一个自己满意得成品吧
  • Python，这个有点蛋疼的，之前的打算是学玩 爬虫框架就开始学 Opencv 的，细细数下来大概快半年没动了，占时打算是工作了用空余时间再研究
  • 关于前端，还是很想学啊，之前毕设用 bootstrap 写了个小站，自己看着觉得还不错，所以没办法了，教练！我…..
  • 然后是吉他，撩妹、装逼全看它了，我不会放弃的！！！

最后

这次博客的样式终于感到满意了，算下来前前后后都已经迁移好几次了，要么是样式不好看，要不是没钱续了，不过这次就好啦，待会再找个好一点的图床，一切就大功告成了 (｡･∀･)ﾉﾞ
