流下了没有技术的泪水

做了一个搜索书签的小插件

发表于 2018-05-31 • 分类于 流下了没有技术的泪水

前言

最近在折腾Chrome Extension，其实也是一时的心血来潮，断断续续学了几天，不多的内容也看得七七八八了。然后就在昨天决定写点小东西出来玩玩，也算是记录一下这几天所学的知识。
当前这个小插件只也是第一版，功能还不是很齐全（其实也没啥功能→_→），计划的是后续优化一下，到时候再说吧，如果有时间的话。毕竟临近毕业，导师检查论文、聚会啥的，麻烦的事也不少。

目前只有 跳转 和 删除 功能，这里用 GIF 简单演示一下吧

优化

由于一些 炫酷 有的没的效果，整个插件显得有些臃肿，主要为了能够离线使用，所以必要的文件我直接避免从CDN上加载。后续将会继续优化一下插件的体积以及一些功能，另外顶部使用的图片没有的得到授权。所以，原本打算发布到 Chrome 应用商店的想法得搁置一下了，目前先将它放到 GitHub 上吧，等后面自己改得满意了在发布~

随笔
