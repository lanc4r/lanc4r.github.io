安利一款 PPT 制作工具

发表于 2019-06-16 • 分类于 疯狂安利 , 工具篇

前言

第一次使用发生在念大学期间，当时是小组内需要使用到 PPT 进行演示，秉着实用(装逼)的原则去网上搜了一下如何做出高逼格的 PPT 文档，于是找到了这篇文章的主角….

其实当时还用过 Prezi 这种类似的东西，也很炫酷，不过好像都没有 3D 的效果，接下来说的这个东西是真的实现了 3D 的效果，并且还能让你体验 代码编写 PPT 的快感(并不快乐)。

Impress.js

Impress.js 是一款基于 Prezi 灵感而诞生的 PPT 框架，具体的东西就不说啦，可以参考它官网的地址，就我而言我觉得它最炫酷的要数 3D 效果啦。项目地址请移步 Impress.js。

Demo

Demo 是我自己的一个 idea，准备做一个时光轴，这里只玩了一会儿它 3D 的效果(大部分原因是前端狗屁不会)，感觉还不错，Demo 地址。

之前是想自己学学前端来完善，但现在发现是真的难学…..

好想找一个前端的小伙伴啊 （；´д｀），一起撸代码，一起实现内心有趣的 idea。

代码

Demo 代码如下，很简单，读读官方文档就可以上手了，最主要的是你的想象力，这个才是关键。

在线编辑

之前发现一个在线编辑网站 ：strut.io，不过发现好像挂了，然后我去 Github 上搜了一下发现一个作者把项目开源了，Strut，需要用到的可以通过 Node.js 本地搭建环境运行。

之前网站还没有挂的时候我体验过，是大学时候帮小组的同学做 PPT，靠着这个 3D 效果成功装逼，不过我记得当时是没有 data-rotate-z 这个属性的，也就是不能按 z轴旋转。
