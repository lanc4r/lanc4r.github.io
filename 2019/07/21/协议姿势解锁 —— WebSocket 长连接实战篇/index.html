记录一下遇到的两个小问题

JavaWeb 添加第三方 Jar 问题

JavaWeb 添加外部的 Jar 包时候，如果想要发布使用的话，光添加到 IDE 当中是没用的(我使用的是 Eclipse)，需要添加到所关联的 Tomcat 的 lib 目录下，至于为什么后面找个机会了解一下，当前可以参考博客 javaWeb项目如何添加jar包

老版本 JavaWeb 项目中 EL 相关问题

老版本的 JavaWeb(2.3版本，IDE 是 Eclipse) 所编写的 JSP 页面中，默认 EL 表达式是关闭的，需要手动开启，参考博客 spring mvc EL ModelAndView的 Model 值 在jsp中不显示
