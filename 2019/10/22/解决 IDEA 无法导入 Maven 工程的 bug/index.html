解决 IDEA 无法导入 Maven 工程的 bug

发表于 2019-10-22 • 分类于 流下了没有技术的泪水 , FAQ

解决 IDEA 无法导入 Maven 工程的 bug

最近公司整体项目迁到了 Git，同时我也换了自己的 IDE，今天在自己玩的时候发现一个问题，发现没有办法使用 Maven 项目了~

环境

  • Java version “1.8.0_111”
  • maven 3.6.2
  • idea 2018.2

报错

当我在设置好 Maven 配置后，IDEA 控制台抛出错误

无法导入 Maven 工程？？于是顺着提示，我开始看 IDEA 输出的日志…

共有两个错误，似乎都是没有办法导入 Maven 的相关类，嗯，很明显我解决不了这个问题，于是开始搜索信息，找到了一个导致此问题的原因：

maven 3.6.2 版本问题

详情可参考下面两篇文章：

Idea unable import maven project
Exception when importing projects with Maven 3.6.2

好吧，接下来去 maven 镜像地址 下载了 3.5.4 版本，再次试着导入，ok，一切正常~

另外，发现 2019.2.3 版本的更新日志中提示已经修复了此问题，行吧，但是懒癌患者是不会折腾的~
